# Projeto MySQL – Loja de CDs

Este projeto foi desenvolvido com o objetivo de praticar a modelagem e implementação de um banco de dados utilizando MySQL.

O sistema representa uma **loja de CDs**, contendo tabelas e relacionamentos que simulam um cenário real de vendas e organização de dados.

## Estrutura do Projeto

O banco de dados é composto por tabelas que representam:
-titulos_pedidos
-titulos
-titulos_artistas
-gravadoras
-categorias
-artistas
-pedidos
-dependentes dos funcionarios (esposas ou maridos)
-funcionarios
-clientes
-conjuge dos clientes (esposas ou maridos)
-cidade
-estado


As tabelas estão relacionadas entre si por meio de **chaves primárias e estrangeiras**, garantindo a integridade dos dados.

## Conceitos Trabalhados

- Criação de banco de dados
- Criação de tabelas
- Relacionamentos entre tabelas
- Chaves primárias e chaves estrangeiras
- Inserção de dados
- Consultas SQL

## Tecnologias Utilizadas

- MySQL
